TradeGuard — Forex Trade Journaling Website
https://tradeguard-20.netlify.app/
A web-based trade journal built to help forex traders log and review their trades. This project was built as part of my self-directed journey into programming, applying HTML, CSS, and JavaScript fundamentals to a practical, real-world tool for traders.

Overview

TradeGuard lets users log individual forex trades — including entry price, exit price, and resulting profit or loss — in a clean, easy-to-scan interface. The goal was to build something genuinely useful for tracking trading performance over time, while practicing core front-end web development skills.

How It Works


Enter Trade Details — User inputs entry price, exit price, and any other relevant trade info.
Calculate P&L — The site calculates profit or loss based on the entered values.
View Trade History — Logged trades are displayed in a list/table for easy review.
